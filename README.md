# dotfiles
dotbot &amp; zinit kickstarter

Using dotbot to kickstart a new machine, zinit to give zsh a charge

## To get started:
```bash
git clone https://github.com/Ogglord/dotfiles.git ~/.dotfiles && cd .dotfiles
./install
exec zsh
```

### Apps & config included:
 - zsh (doh!)
 - fzf
 - dust
 - jq
 - tmux
 - alacritty
 - a few nerd fonts
